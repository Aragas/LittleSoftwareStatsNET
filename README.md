# SystemInfoLibrary


####Windows:
+ **CPU:**  
	- Supports only one physical CPU.
+ **GPU:**  
	- Supports only one physical GPU.
+ **RAM:**  
	- Full support.  
	
####Unix:  
+ **CPU:**  
	- Supports multiple physical CPUs.
+ **GPU:**  
	- Supports only one physical GPU. Partial support. *glxinfo* is needed to be installed.
+ **RAM:**  
  - Full support.  
  
####MacOSX:  
+ **CPU:**  
	- Supports only one physical CPU.
+ **GPU:**  
	- Not supported.
+ **RAM:**  
	- Partial support.
